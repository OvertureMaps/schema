# overture-schema-validation

Validation command for the Overture Maps schema CLI.

This package provides the `validate` subcommand for the `overture-schema` CLI tool.

## Installation

This package is automatically loaded by `overture-schema-cli` via entry points.

```bash
pip install overture-schema-cli overture-schema-validation
```

## Usage

```bash
overture-schema validate <file>
```

See the main [overture-schema](https://github.com/OvertureMaps/schema) documentation for more details.
