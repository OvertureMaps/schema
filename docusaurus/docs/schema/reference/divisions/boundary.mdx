# Boundary

A boundary is a border between [divisions](/schema/reference/divisions/division).

A boundary is typically an officially-defined border
between two [divisions](/schema/reference/divisions/division). In light of
geopolitical issues, some boundaries represent standalone
disputed or treaty lines and therefore do not coincide with the border
of a division.
